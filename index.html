Proyecto Destacado

GEDANI – Digitalización de Informes en Terreno

GEDANI es una aplicación web desarrollada como proyecto de título para resolver la ineficiencia en la recopilación manual de informes en terreno. Los inspectores registraban datos a mano, generando pérdida de información, errores de transcripción y reportes incompletos. La solución digitaliza el proceso mediante formularios estructurados, almacenamiento en Firebase y autenticación segura. Permite centralizar la información en tiempo real, mejorando trazabilidad y reduciendo ambigüedades en los reportes.

React JavaScript Firebase HTML CSS

Impacto esperado:

  • Reducción de errores de transcripción
  • Centralización de datos en tiempo real
  • Mejora en trazabilidad de informes
